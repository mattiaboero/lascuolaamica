Accesso con Token

Questa pagina usa accesso privato con token. Primo accesso con link #k=..., poi nello stesso browser puoi usare anche l'URL normale.

Configurazione token non valida. Aggiorna admin/editor-config.js con un hash SHA-256 prima dell'uso.

Suggerimento: usa il token nel frammento URL (hash) per ridurre l'esposizione nei log: .../admin/esercizi#k=IL_TUO_TOKEN.
